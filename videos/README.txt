# Convenção de nomes — /videos/

Cada vídeo usado nas páginas de case (Reels e VSL) deve seguir o padrão:

  {cliente}-{tipo}-{numero}.mp4

Exemplos já referenciados no template (case-sections-preview.html):
  hub4-reel-01.mp4
  hub4-reel-02.mp4
  hub4-reel-03.mp4
  hub4-vsl.mp4

Basta colocar os arquivos .mp4 reais aqui com esse nome exato que eles
passam a tocar automaticamente no lugar do poster (imagem de capa).

Formato recomendado: .mp4 (H.264), otimizado para web (HandBrake ou
similar) para não pesar o carregamento da página.
